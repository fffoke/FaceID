# 🚀 Инструкция по использованию Лаунчера

## Быстрый старт

1. **Запустите лаунчер:**
   - Двойной клик на файл `launcher.bat`
   - Или через командную строку: `python launcher.py`

2. **В открывшемся окне вы можете:**
   - Запускать/останавливать Java Web приложение (панель)
   - Запускать/останавливать Python скрипты KPP1 и KPP2
   - Изменять IP адреса камер
   - Просматривать логи всех процессов

## Первый запуск Java приложения

### Вариант 1: Через Maven (если Maven установлен)
Лаунчер автоматически запустит приложение через `mvn spring-boot:run`

### Вариант 2: Через JAR файл (рекомендуется)
Если Maven не установлен или вы хотите использовать JAR:

1. **Откройте проект в IntelliJ IDEA**
2. **Соберите JAR файл:**
   - Откройте панель Maven (справа в Idea)
   - Разверните `Lifecycle`
   - Дважды кликните на `package`
   - Дождитесь завершения сборки
3. **JAR файл будет создан в папке `target/`**
4. **Лаунчер автоматически найдет и использует JAR файл**

## Использование

### Запуск компонентов:
1. Нажмите кнопку **"▶ Запустить"** для нужного компонента
2. Статус изменится на "Запущен" (зеленый цвет)
3. Логи будут отображаться в нижней панели

### Остановка компонентов:
1. Нажмите кнопку **"⏹ Остановить"** для нужного компонента
2. Статус изменится на "Остановлен" (красный цвет)

### Изменение IP адресов камер:
1. Введите новый IP адрес в поле "IP адрес камеры" для KPP1 или KPP2
2. Нажмите кнопку **"💾 Сохранить IP"**
3. Изменения будут сохранены в файлах `face_kpp1.py` или `face_kpp2.py`
4. **Важно:** После изменения IP перезапустите соответствующий скрипт

## Требования

- ✅ Python 3.x (должен быть в PATH)
- ✅ Java (для запуска Java приложения)
- ✅ Maven (опционально, если хотите запускать через Maven)
- ✅ Настроенный venv в `OpenCVRework/venv/`

## Примечания

- Лаунчер автоматически использует Python из `venv`, не нужно активировать его вручную
- Все процессы запускаются в фоновом режиме
- При закрытии лаунчера все запущенные процессы будут остановлены
- Логи можно очистить кнопкой "Очистить логи"

## Устранение проблем

### "Maven не найден"
- Установите Maven и добавьте в PATH
- Или соберите JAR файл через Idea (см. выше)

### "Python из venv не найден"
- Убедитесь, что папка `OpenCVRework/venv/` существует
- Если venv не настроен, создайте его:
  ```bash
  cd OpenCVRework
  python -m venv venv
  venv\Scripts\activate
  pip install -r requirements.txt  # если есть requirements.txt
  ```

### "Java приложение не запускается"
- Проверьте, что порт 8080 свободен
- Убедитесь, что PostgreSQL запущен (если используется)
- Проверьте логи в лаунчере для деталей ошибки
