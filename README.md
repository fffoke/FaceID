# FacePanel - Система распознавания лиц и контроля посещений

## 📋 Описание проекта

**FacePanel** - это комплексная система для распознавания лиц и контроля посещений на мероприятиях. Система состоит из двух основных компонентов:

- **Spring Boot Backend** - серверная часть для обработки данных распознавания
- **Python OpenCV Script** - клиентская часть для захвата видео и распознавания лиц

## 🏗️ Архитектура системы

### Backend (Spring Boot)
- **Framework**: Spring Boot 3.x
- **Database**: PostgreSQL
- **ORM**: JPA/Hibernate
- **Template Engine**: Thymeleaf
- **Real-time Communication**: WebSocket + STOMP
- **Build Tool**: Maven

### Frontend
- **Templates**: Thymeleaf HTML
- **Styling**: Custom CSS (Dark Theme)
- **Real-time Updates**: JavaScript + SockJS + STOMP
- **Responsive Design**: Grid Layout

### Python Script
- **Computer Vision**: OpenCV (cv2)
- **Face Recognition**: Custom face detection algorithms
- **HTTP Client**: requests library
- **Image Processing**: PIL/Pillow

## 🎯 Основной функционал

### 1. Распознавание лиц в реальном времени
- Захват видео с камеры
- Детекция и распознавание лиц
- Отправка данных на сервер через REST API
- WebSocket уведомления для обновления интерфейса

### 2. Управление мероприятиями
- Создание и завершение мероприятий
- Автоматическое создание "Default" сессии при первом распознавании
- Контроль посещений в реальном времени
- Статистика участников

### 3. Управление персонами
- CRUD операции для персон
- Связь с файлами изображений лиц
- Автоматическое обновление списков через WebSocket

### 4. История и аналитика
- Детальная история мероприятий
- Время прихода каждого участника
- Статистика посещаемости
- Фильтрация по датам

## 🗄️ Структура базы данных

### Основные сущности:
- **Person** - информация о персонах (имя, фамилия, должность, файл фото)
- **Session** - мероприятия/сессии (название, время начала/окончания, статус)
- **Attendance** - записи о посещениях (персона, сессия, время, тип события)

### Связи:
- Person ↔ Attendance (One-to-Many)
- Session ↔ Attendance (One-to-Many)

## 🔄 Поток данных

1. **Python скрипт** захватывает видео и распознает лица
2. **REST API** получает данные распознавания
3. **AttendanceService** создает записи о посещениях
4. **WebSocket** отправляет уведомления клиентам
5. **Frontend** обновляется в реальном времени
